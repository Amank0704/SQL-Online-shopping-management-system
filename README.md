# Online Shopping Management System

A MySQL-based database project developed during Campus Recruitment Training (CRT). This project simulates an online shopping platform and demonstrates the implementation of relational database concepts for managing customers, products, orders, invoices, and payments.

## Features

* Customer Management
* Product & Category Management
* Order Processing
* Invoice Generation
* Payment Tracking
* Data Retrieval and Reporting using SQL Queries

## Technologies Used

* MySQL
* SQL
* DBMS Concepts

## Database Tables

* Customer
* Category
* Product
* Orders
* Order_Items
* Invoice
* Payment

## SQL Concepts Covered

* DDL Commands (CREATE, ALTER, DROP)
* DML Commands (INSERT, UPDATE, DELETE)
* SELECT Queries
* Filtering and Sorting
* Aggregate Functions
* GROUP BY and HAVING
* Joins
* Primary Keys and Foreign Keys

## Project Purpose

The objective of this project is to apply database design principles and SQL operations in a real-world e-commerce scenario. It was completed as part of Campus Recruitment Training (CRT) to strengthen practical database management and query-writing skills.
